分工安排
## 组长赵灿：主程序（核心逻辑+战斗系统）
**负责：所有游戏底层逻辑、战斗、职业、升级、数值**
### 工作内容
1. 角色基础框架
	- 玩家实体制作、血量/蓝量/攻击力/防御/速度属性系统
	- **三职业切换系统**：初始剑士、游侠（高速远程）、盾骑士（高防近战）
	- 角色等级经验系统、升级自动加属性、升级弹窗提示
2. 战斗系统（项目核心）
	- 地下城遇敌触发战斗
	- 普攻+职业专属技能编写
	- 怪物AI、怪物血量、掉落物品逻辑
	- 战斗胜负判定、战后经验结算
3. 数据与存档
	- 全局游戏数据管理
	- 角色等级、职业、血量、道具**本地存档读档**
4. 通用工具脚本
	- 血条、蓝条、经验条UI逻辑
	- 伤害数字飘字、暴击、闪避判定

## 组员许世豪：场景搭建+地图+碰撞+关卡设计
**负责：村庄全场景、地下城多层地图、地图交互、寻路、场景跳转**
### 工作内容
1. 像素瓦片地图制作
	- 完成**村庄完整场景**铺设（房屋、道路、水井、草地、出入口）
	- 设计**多层地下城地图**（第一层小怪、中层精英、深层BOSS房）
	- 所有地图瓦片碰撞层、行走区域、阻挡区域设置
2. 场景功能实现
	- 村庄 ↔ 地下城 **场景传送跳转**
	- 地图触发区域（进入副本、离开副本）
	- 角色地面行走、地图视角相机跟随
3. 环境与氛围
	- 场景装饰物摆放、植被、建筑摆放
	- 地下城黑暗氛围、光源简单效果
4. 简单NPC点位布置
	- 村庄商人、旅馆、任务NPC固定位置预留

## 组员张凌丰：UI界面+美术资源+交互+剧情文案
**负责：所有界面、图标、动画、文案、音效、NPC对话**
### 工作内容
1. 全套游戏UI制作
	- 主界面、角色状态栏、背包界面
	- 职业选择界面、升级界面、战斗技能栏
	- 商店购买界面、休息回血界面
2. 动画与美术适配
	- 玩家三职业行走/待机/攻击像素动画导入适配
	- 怪物攻击、死亡动画调试
	- 技能特效、普攻特效制作
3. 剧情与交互
	- 村庄NPC对话系统编写
	- 旅馆休息回血、商店买卖交互逻辑对接程序
	- 游戏提示文字、任务文案、世界观文案
4. 音效与体验优化
	- 行走、攻击、升级、打怪音效挂载
	- 游戏弹窗提示、新手引导简易文字
